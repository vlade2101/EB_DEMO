|--------|-----|

| Tamna pozadina (proizvodi) | `#0A0A0A` |

## Pokretanje

Otvorite `index.html` u pregledniku (dvostruki klik ili Live Server u VS Code/Cursor).

Za lokalni server (opcionalno):

```bash
cd eurobeton-web
npx --yes serve .
```

## Struktura

- `index.html` – početna stranica
- `css/variables.css` – boje i varijable
- `css/style.css` – stilovi
- `js/main.js` – izbornik, slider, kolačići
- `assets/logo/` – logotip
- `assets/icons/` – zastavice jezika

## Zamjena slika

Zamijenite URL-ove u `index.html` (hero, pločice, proizvodi) datotekama u `assets/images/`.

## Napomene

- Sekcija **„Betonski proizvodi visoke kvalitete”** koristi tamni EUROBETON dizajn (crna pozadina, bijeli krugovi).
- Ostatak stranice slijedi svijetliji, profesionalni layout (hero preko cijelog zaslona, plavi akcenti).
- Linkovi na podstranice trenutno vode na `#` – nadogradnja u sljedećim fazama.
